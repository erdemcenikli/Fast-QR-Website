About Link QR code reader and barcode scanner

Link is an iPhone QR code reader and barcode scanner app that recognizes QR codes, EAN-13/8, UPC-A/E, Code 128, Code 39, Data Matrix and PDF417. It opens website links, adds Wi-Fi networks, saves vCards/contacts, copies text, and looks up products by barcode — all on-device, with no ads. If you searched for "QR code scanner iPhone", "barcode scanner for iOS", or "QR reader app without ads", this is built for you.

Frequently asked questions

Does Link work offline?

Yes. Scanning runs entirely on your iPhone for common formats.

Which codes are supported?

QR, EAN-13/8, UPC-A/E, Code 128, Code 39, Data Matrix, PDF417, and more.

Is my data private?

On-device processing means images are not sent to servers. You control optional history.

Is Link free?

Yes. Link is free to download. Some advanced features may require in-app purchase if you add them later.
